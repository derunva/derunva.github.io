Git - сиситема контролю версій

  • Встановіть консольний інтерфейс Git
  • Зареєструйтесь на GitHub
  • Створіть новий новий репозиторій
  • За інструкциєю на GitHub з'єднайте зі своїм проектом з CV

Необхідні команди

  • git init - команда для ініціалізації нового гіт репозиторю
  • git stauts - перевірка стану репозиторія
  • git add . - додати всі файли до змін
  • git commit -m "коментар до змін" - оголосити зміни з коментарем
  • git push origin master - завантажити зміни до віддаленого репозиторію
  • git pull - зтягнуті зміни з віддаленого репозиторію
  • git remote add origin https://github.com/user/repo.git - додати віддалений репозиторій

Завжди уважно читаєм що пише консоль !!!

Навігація в консолі

  • D: - перейти до дистку D
  • dir(for Windows) / ls (Mac / Linux) - лістинг поточної директорії
  • cd *назва теки* - перейти до теки
  • mkdir *назва теки* - створення теки
  • touch *назва фалу* (Mac / Linux) - створення файлу
  • *назва програми* *файл* --*опції запуску* - запуск програмою файлу / запуск програми, в залежності від програми виклик може трохи різнитися

Де взяти консоль?

  • Unix - ctrl + Alt + T або шукайте програму Terminal
  • Windows - window + r та пропишіть cmd, для роботи з git використовуйте git Bash
